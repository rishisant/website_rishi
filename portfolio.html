Atelier

       Atelier provides a unique and convenient platform for artists to showcase their work, and for art lovers to discover new and exciting artists and artworks. With Atelier, everyone can appreciate and support the arts in a fun and easy way. I founded Atelier for the sole purpose of removing the barriers that prevent artists from sharing their work with the world, as well as improve feedback mechanisms such as like buttons, substituting them for a more meaningful and engaging voting system. Atelier is currently in development, and I am the sole developer. I am using React.js for the frontend, and Node.js for the backend.


Check it out here: Atelier Website


WordChain

       WordChain is a game about chaining words together. A chain is denoted by two or more words which both connect through sharing similar letters. You are given a topic, such as countries, in which your first word must be the name of a Country. Say you answer with Brazil. The next word must start with the last letter of the previous word, so in this case, 'l' must be the first character of the second word.

NamuDB

       NamuDB is a program that allows you to store and retrieve data from a database. It creates a database file, which you can then add, remove, and edit data from. For data security, all .namu files are encrypted with a custom-generated encryption key, unique to the .namu database. The encryption key is stored in the same .namu file and is also encrypted with another unique key; this double encryption makes the data in the .namu file virtually impossible to decrypt without the key.


Check out the Git Repo here: GitHub Link


Population Simulation: A Study on Gene Variance

       I created a sample organism in Python and programmed it to live in an environment. This organism's only goal is to find food in order to survive and reproduce. I added genetic variance within this environment, with each gene giving an organism a certain ability to aid it in survival. This study highlighted which genes survive in certain environments and which genes could not.
