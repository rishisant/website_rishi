Here are some of the projects I've worked on in the past year.

WordChain WordChain is a game about chaining words together. A chain is denoted by two or more
words which both connect through sharing similar letters. You are given a topic, such as
countries, in which your first word must be the name of a Country. Say you answer with
Brazil. The next word must start with the last letter of the previous word, so in this
case, 'l' must be the first character of the second word.

Eigood Eigood is a project I started in order to teach casual/colloquial English to Japanese speakers.
Eigood contains custom English lessons, tailored specifically for Japanese native speakers.
I got the chance to interact with my Japanese user base and created multiple lessons that are
within the application itself. I created Eigood with Swift and SwiftUI through XCode.
World of Wreckage The World of Wreckage: Analyzing Salvaged Vehicles is a project completed during TAMU Datathon 2021.
Me and my team analyzed salvaged vehicles, a good substitute due to the expensive used car market.
This was our first time ever web-scraping and we were able to make a pretty solid analysis. I also got to work
on an algorithm that would evaluate the estimated profit of a car, based on a plethora of variables such as brand,
make, model, damage type, damage severity, odometer readings, and more!

Check out our devpost: Devpost Link

Custom Discord Bot I created a custom discord bot for my friend's server! This was my first time ever creating a custom Discord bot
so there were quite a few challenges. I made the bot in Python. Here is my favorite command (!!zingo) running on
my test server a while ago. The bot has a few commands it can run, such as !!number x y, which starts a
number guessing game from 0 to x with y number of guesses. It also has commands such as !!gasmeup which gives the
user a nice compliment upon invocation.
Empire Simulator Empire Simulator was the first web project I had taken on. My goal was to create a simulator game in
which you create your own empire. There are three playable archetypes, each with varying strengths.
This game was never fully completed, however, I got a decent amount of the game mapped out already.
